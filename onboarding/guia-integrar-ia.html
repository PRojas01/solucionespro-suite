1

Instala Node.js

Es la base que necesitan los asistentes. Se instala una sola vez.

Descargar Node.js para Windows ↗

Abre el archivo descargado y pulsa Siguiente hasta terminar. Si te ofrece “Add to PATH”, déjalo marcado.

Descargar Node.js para Mac ↗

Abre el archivo .pkg descargado y pulsa Continuar hasta terminar.

Pega esto en la terminal curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash - && sudo apt install -y nodejs

Te pedirá tu contraseña. Al escribirla no se ve nada en pantalla — es normal, escribe y pulsa Enter.

Para comprobar que funcionó Abre la terminal y pega node --version. Si aparece un número como v22.14.0, quedó listo.
¿Cómo abro la terminal?

Windows: pulsa la tecla Windows, escribe PowerShell y ábrelo.

Mac: pulsa Cmd + barra espaciadora, escribe Terminal y Enter.

Linux: pulsa Ctrl + Alt + T.

Verás una ventana con texto y un cursor. Ahí pegas los comandos con Ctrl+V (Cmd+V en Mac) y pulsas Enter.
